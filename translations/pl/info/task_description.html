Żaba skacząca po nieskończonej dwuwymiarowej siatce liczb całkowitych jest reprezentowana jako 4-krotność postaci {% if is_js %}[x, y, dx, dy]{% else %}(x, y, dx, dy){% endif %} gdzie x, y jest pozycją początkową żaby w czasie t = 0, a dx, dy jest jej stałym wektorem kierunku dla każdego skoku. Czas przesuwa się w dyskretnych krokach całkowitych, tak że każda żaba wykonuje jeden skok przy każdym tyknięciu zegara.

Biorąc pod uwagę dwie żaby frog1, frog2, które początkowo stoją na różnych kwadratach, proszę zwrócić czas, w którym obie żaby wskoczą na ten sam kwadrat. Jeśli te dwie żaby nigdy nie wskoczą do tego samego kwadratu w tym samym czasie, funkcja powinna zwrócić {% if is_js %}null{% else %}None{% endif %}.

Proszę spojrzeć na przykład {% if is_js %}[0, 0, 0, 2], [0, 10, 0, 1]{% else %}(0, 0, 0, 2), (0, 10, 0, 1){% endif %}. X są równe i nie zmieniają się (dx1 == dx2 == 0), więc możemy je zignorować.

Państwa funkcja nie powinna zawierać żadnych pętli, ale wynik powinien być obliczany za pomocą instrukcji warunkowych i arytmetyki liczb całkowitych.

Dane wejściowe: Dwie krotki po cztery liczby całkowite {% if is_js %}number{% else %}int{% endif %}.
