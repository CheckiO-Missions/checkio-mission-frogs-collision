Жаба, яка стрибає по нескінченній двовимірній сітці цілих чисел, подається у вигляді 4-х кортежів виду {% if is_js %}[x, y, dx, dy]{% else %}(x, y, dx, dy){% endif %} де x, y - початкова позиція жаби у момент часу t = 0, а dx, dy - вектор її постійного напрямку для кожного стрибка. Час йде дискретними цілочисельними кроками так, що кожна жаба робить один стрибок за кожний такт годинника.

За заданими двома жабами frog1, frog2, які спочатку стоять на різних клітинках, поверніть час, коли обидві жаби стрибнуть в одну і ту ж клітинку. Якщо ці дві жаби ніколи не стрибають в одну і ту ж клітинку одночасно, то функція повинна повернути {% if is_js %}null{% else %}None{% endif %}.

Розглянемо приклад {% if is_js %}[0, 0, 0, 2], [0, 10, 0, 1]{% else %}(0, 0, 0, 2), (0, 10, 0, 1){% endif %}. X рівні і не змінюються (dx1 == dx2 == 0), тому ми можемо їх ігнорувати.

Ваша функція не повинна містити жодних циклів, але результат має обчислюватися за допомогою умовних операторів та цілочисельної арифметики.

Вхідні дані: Два кортежі по чотири цілих числа {% if is_js %}number{% else %}int{% endif %}.

Вихідні дані: Ціле число або {% if is_js %}null{% else %}Немає{% endif %}.
